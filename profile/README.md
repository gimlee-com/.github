## 🏴‍☠️ What is Gimlee?

Gimlee is a **decentralized marketplace** that connects buyers and sellers for direct peer-to-peer commerce using cryptocurrency. No banks, no payment processors, no custody of your funds — ever.

### Trustless Payment Verification

Sellers import a **viewing key** (read-only) into the platform. Gimlee verifies payments directly on the blockchain without ever taking custody of funds. This means:

- ✅ **Non-custodial** — your keys, your coins
- ✅ **Transparent** — payments verified on-chain
- ✅ **Privacy-preserving** — compatible with privacy-focused cryptocurrencies

## 🚀 Vision

The long-term goal is a **community-driven, privacy-centric economy** that features:

- **Decentralized data storage** — enabling independent, self-synchronizing Gimlee instances
- **Self-hosted mode** — run your own store with a single Docker image
- **Modular crypto clients** — blockchain integrations extracted as standalone libraries for the community
